# el表达式

会从四大域对象中获取属性，语法`${}`

没有指定作用域的话，会由小到大的范围搜索，即先从page，再到request，再到session,最后时application

```jsp
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>Title</title>
</head>
<body>
    ${username}
    ${pageScope.username}
    ${requestScope.username}
    ${sessionScope.username}
    ${applicationScope.username}
</body>
</html>
```
